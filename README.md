# Customer Complaint Classification System

## Description
This project implements an automated customer complaint classification system that categorizes complaints into predefined classes such as Billing, Service, and Technical Issues. The system uses TF-IDF vectorization and Logistic Regression to classify complaints and route them to the appropriate department.

---

## Problem Statement
Organizations receive large volumes of customer complaints daily. Manual categorization and routing of complaints is inefficient and time-consuming.

This project automates complaint classification for faster handling and routing.

---

## Objective
- Classify customer complaints into predefined categories  
- Route complaints to the correct department automatically  
- Improve complaint handling efficiency  
- Demonstrate practical NLP text classification workflow  

---

## Categories Supported
- Billing  
- Service  
- Technical  

---

## Methodology
1. Collect labeled complaint dataset  
2. Preprocess complaint text  
3. Convert text into TF-IDF vectors  
4. Train Logistic Regression classifier  
5. Predict complaint category  
6. Map category to department  
7. Display classification result  

---

## Machine Learning Workflow
- Text Preprocessing  
- TF-IDF Vectorization  
- Supervised Classification  
- Department Mapping
